# WakeUpp - Умный будильник для студентов

Мобильное приложение-будильник для студентов с автоматическим расчетом времени подъема на основе расписания, маршрута и погоды.

## Возможности

- Автоматическая загрузка расписания с сайта университета (rasp.dmami.ru)
- Расчет времени будильника с учетом:
  - Времени утренней рутины
  - Времени в пути до университета
  - Пробок на дорогах
  - Запаса времени
- Прогноз погоды и рекомендации (что надеть, взять ли зонт)
- Построение маршрута до университета
- Кэширование данных для офлайн работы

## Технологии

- **React Native** + **Expo** - кроссплатформенная разработка
- **React Navigation** - навигация между экранами
- **AsyncStorage** - локальное хранилище
- **Яндекс.Карты API** - построение маршрутов
- **OpenWeatherMap API** - данные о погоде

## Установка и запуск

### Требования
- Node.js 14+
- npm или yarn
- Expo Go (для тестирования на телефоне)

### Установка зависимостей

```bash
npm install
```

### Запуск приложения

```bash
# Запуск в режиме разработки
npm start

# Запуск на Android
npm run android

# Запуск на iOS
npm run ios

# Запуск в веб-браузере
npm run web
```

## Настройка API ключей

### 1. OpenWeatherMap (погода)

1. Зарегистрируйтесь на https://openweathermap.org/api
2. Получите бесплатный API ключ
3. Откройте `api/weather.js`
4. Замените `YOUR_API_KEY_HERE` на ваш ключ:

```javascript
const API_KEY = 'ваш_ключ_здесь';
```

### 2. Яндекс.Карты (маршруты)

1. Зарегистрируйтесь на https://developer.tech.yandex.ru/
2. Создайте приложение и получите API ключ для "JavaScript API и HTTP Геокодер"
3. Откройте `api/routes.js`
4. Замените `YOUR_YANDEX_API_KEY` на ваш ключ:

```javascript
const YANDEX_API_KEY = 'ваш_ключ_здесь';
```

**Примечание:** Приложение работает и без API ключей, используя тестовые данные.

## Использование

### Первый запуск

1. Откройте приложение
2. Перейдите в раздел **Настройки**
3. Заполните необходимые данные:
   - Номер группы (например: `151-331`, `231-324`)
   - Домашний адрес
   - Адрес университета
   - Время утренней рутины (сколько нужно на сборы)
   - Способ передвижения (общественный транспорт, авто, пешком)
   - Дополнительное время (запас)
4. Нажмите **Сохранить настройки**

### Просмотр расписания

1. Перейдите в раздел **Расписание**
2. Расписание загрузится автоматически
3. Выберите день недели для просмотра занятий
4. Для обновления нажмите **Обновить расписание**

### Маршрут

1. Перейдите в раздел **Маршрут**
2. Маршрут построится автоматически на основе адресов из настроек
3. Вы увидите:
   - Время в пути
   - Шаги маршрута
   - Информацию о пробках
   - Рекомендуемое время выезда

### Главный экран

На главном экране отображается:
- **Следующий будильник** - автоматически рассчитанное время подъема
- **Погода** - прогноз погоды на утро
- **Рекомендации** - что взять с собой, как одеться
